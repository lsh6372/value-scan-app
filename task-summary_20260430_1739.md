# 顶部导航栏开发

## 目标
为已有三个页面开发顶部导航栏，点击切换页面。

## 实现
1. 创建 `src/components/layout/MainLayout/index.jsx`
   - 使用 Ant Design Layout + Menu 组件
   - Logo 区域（渐变色 ValueScan 标题）
   - 导航菜单（首页、代币列表、资金异动）
   - 内容区域

2. 更新 `App.jsx`
   - 包裹 MainLayout
   - 新增 Menu 组件主题配置

3. 导航菜单配置
   - 首页 `/` - HomeOutlined
   - 代币列表 `/tokens` - UnorderedListOutlined
   - 资金异动 `/long-inflow` - RiseOutlined

## 新增页面流程
1. 在 `router.jsx` 添加路由
2. 在 `MainLayout/index.jsx` 的 `menuItems` 数组添加菜单项

## 结论
构建通过，顶部导航栏已生效。
