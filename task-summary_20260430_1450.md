# 代币列表页面开发

## 目标
实现代币列表页面，支持搜索和刷新功能。

## API 信息
- 接口：POST /open/v1/vs-token/list
- 参数：search（可选，模糊搜索）
- 返回：{ id, symbol, name }[]

## 输出文件
- src/api/services.js - getTokenList API 方法
- src/pages/TokenList/index.jsx - 代币列表页面（搜索 + 刷新）
- src/vite.config.js - 添加 @ 路径别名

## 结论
构建成功。页面包含搜索框和刷新按钮，显示 ID/符号/名称三列。
