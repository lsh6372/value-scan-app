# 大盘分析页面开发

## 需求
1. 新页面名称"大盘分析"，路由 `/market-analysis`
2. 移植 trader 文件夹的 SSE 订阅功能（market_subscribe.py）
3. 添加"启动订阅"和"终止订阅"按钮
4. 数据展示在页面，不写入 state.json

## 移植内容（从 market_subscribe.py）
- SSE URL: `https://stream.valuescan.ai/stream/market/subscribe`
- 签名算法: HMAC-SHA256(secretKey, timestamp + nonce)
- 信号解析: 正则 `BTC分析...AI综合分析：利多/利空/震荡`
- 消息格式: `event: market` + `data: JSON`

## 新增文件
1. `src/api/sse-sign.js` - SSE 专用签名工具
2. `src/pages/MarketAnalysis/index.jsx` - 大盘分析页面

## 修改文件
1. `src/router.jsx` - 添加 `/market-analysis` 路由
2. `src/components/layout/MainLayout/index.jsx` - 添加"大盘分析"菜单项（LineChartOutlined 图标）
3. `TOOLS.md` - 更新页面清单和 API 清单

## 页面功能
- 顶部控制栏：标题 + 连接状态 Badge + 启动/终止按钮
- BTC / ETH 信号卡片：利多(绿) / 利空(红) / 震荡(黄)
- 消息日志：最多 50 条，显示原始内容 + 信号标签 + 时间
- 自动重连：连接断开后指数退避重连（最多 60s）

## 注意事项
- VITE_API_KEY 和 VITE_SECRET_KEY 需在 .env.local 中配置
- SSE 使用原生 EventSource API，无需额外依赖
- 与 HTTP API 公用一套 AK/SK

## 结论
构建通过，大盘分析页面已完成。
