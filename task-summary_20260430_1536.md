# 资金异动看涨监控页面开发

## 目标
开发资金异动看涨监控页面，支持刷新功能。

## API 信息
- 接口：POST /open/v1/fund-movement/long-inflow/list
- 返回字段：代币信息、价格、涨幅、24h变化、看涨情绪、交易数、市值等

## 页面功能
- 搜索框：按代币符号搜索
- 刷新按钮：重新加载数据
- 表格展示：代币、交易类型、价格、涨幅、24h变化、看涨情绪、交易数、市值、更新时间
- Alpha/Fomo 标签标识
- 涨跌颜色区分

## 输出文件
- src/api/services.js - getLongInflowList API 方法
- src/pages/LongInflow/index.jsx - 资金异动看涨监控页面
- src/App.jsx - 更新为显示新页面

构建成功。页面已实现数据展示和刷新功能。
